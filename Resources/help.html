Функции приложения

Приложение представляет собой текстовый редактор с возможностью лексического анализа текста. Оно поддерживает основные операции с текстом, управление шрифтами, работу с несколькими документами одновременно и включает справочную систему.

Описание основных пунктов меню

Пункт «Файл» меню текстового редактора

В пункте «Файл» меню текстового редактора реализован следующий функционал:

  • Создание документа
  • Открытие документа
  • Сохранение текущих изменений в документе
  • Сохранение документа в новый файл
  • Выход из документа
↑ Наверх
Пункт «Правка» меню текстового редактора

В пункте «Правка» меню текстового редактора реализован следующий функционал:

  • Отмена изменений
  • Повтор последнего изменения
  • Вырезать текстовый фрагмент
  • Копировать текстовый фрагмент
  • Вставить текстовый фрагмент
  • Удалить текстовый фрагмент
  • Выделить все содержимое документа
↑ Наверх
Пункт «Пуск» меню текстового редактора

При нажатии на пункт «Пуск» происходит запуск лексического анализатора текста.

↑ Наверх
Пункт «Справка» меню текстового редактора

Приложение имеет справочную систему, запускаемую командой «Вызов справки». Справка содержит описание всех реализованных функций меню.

↑ Наверх
Пункт «Шрифт» меню текстового редактора

Интерфейс позволяет пользователю выбирать шрифт, с которым он хочет работать.

  • Копирование текстового фрагмента
  • Вырезать текстовый фрагмент

  • Запуск лексического анализатора
