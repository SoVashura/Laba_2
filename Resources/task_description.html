1. Постановка задачи

Структура – это мощный инструмент языка C, позволяющий создавать собственные составные типы данных. Она объединяет разнородные данные (целые числа, строки, указатели и даже другие структуры) в единую логическую запись, что делает код чище, гибче и удобнее. Объявляется структура с помощью ключевого слова struct.

Формат записи:

struct название_структуры {
    тип поле1; 
    тип поле2; 
    …;
};

Примеры:

1. Обычная структура – с полями:

struct str1 {
    int i; 
    char c; 
    float f;
};

2. Пустая структура – без полей:

struct str1 {
};

Согласно разработанной автоматной грамматике G[‹Start›] лексический анализатор (сканер) структуры будет считать верными следующие записи:

  • struct Address { char city; char street; int house; };
  • struct Empty { };
